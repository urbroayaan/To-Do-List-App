# To-Do List App

A simple console-based to-do list manager built in Python.  
It stores tasks in a text file so they remain saved even after closing the program.

## Features
- Add task  
- List all tasks  
- Mark task as complete  
- Delete task  
- Persistent storage using `task.txt`

## Task Format


`status` is:
- `0` → not completed  
- `1` → completed  

## How to Run
```bash
python to-do_list.py
